# AUTOSAR XSD Schema Files

This directory contains official AUTOSAR XSD schema files for version 00051.

## Required Files

Please download the official XSD files from the AUTOSAR website:
https://www.autosar.org/standards/classic-platform

Required files:
- AUTOSAR_00051.xsd (main schema file)
- xml.xsd (XML namespace definitions)

## Installation

1. Visit https://www.autosar.org/standards/classic-platform
2. Find the release corresponding to this schema version
3. Download the XSD schema files
4. Place them in this directory
